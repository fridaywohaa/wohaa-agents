# Voice assistant lessons (umbrella case)

Date: 2026-03-04

Context: Luke asked a 생활化 question: "今晚出街要唔要拎遮?" after earlier asking temps at 7pm/10pm.

Lessons learned:
1) Use "穩陣模式" for daily-life questions: give conservative, immediately actionable advice without over-questioning.
2) Don't push the user to supply extra details when context already implies the scenario (tonight + outside + earlier 7/10).
3) Make recommendations based on cost/risk asymmetry: carrying a foldable umbrella is low cost; getting caught in rain is high cost.
4) In voice UX, prioritize intent inference first (go out / what to bring) then answer. Ask follow-ups only if it materially changes the decision.

Default policy for similar future questions:
- If user asks "要唔要帶遮" without precise time/location, default to: "帶摺遮/輕便雨褸" + 1-line reasoning.
